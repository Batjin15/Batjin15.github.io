Сайхан вэб загвар — өөрийнхөө код

Энэ хэсэгт богино тайлбар, зорилго, товч агуулга байрлана. Даалгаварт заасан загварын hero хэсгийг энд харуулна.

Сурч эхлэх

Responsive

Гар утас, таблет, компьютерийн дэлгэц дээр тааруулж ажиллана.

Clean UI

Энгийн, цэвэр дизайн ба уншигдахуйц типографи.

Embed Video

Та YouTube эсвэл Google Slides-ийг iframe-ээр оруулж болно.

Гол агуулга

Энд нийтлэл, тайлбар эсвэл том контент байрлана.

Видео (Google Slides / YouTube)

Доорх iframe-ийг өөрийн embed src-тай сольж ашиглана уу.

Хэрвээ YouTube видео оруулах бол доорх iframe-ийг ашиглана: 
<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
